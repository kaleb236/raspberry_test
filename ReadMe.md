# Install audio dependencies
pyaudio:
```bash
sudo apt install python3-pyaudio python3-pip portaudio19-dev
```
Create new virtual env:
```bash
python3 -m venv env
source env/bin/activate
```
install pyaudio:
```bash
pip3 install pyaudio
```
install numpy:
```bash
pip3 install numpy
```
install matplotlib:
```bash
pip3 install matplotlib
```
